# Proyecto Plataformas de Desarrollo - TP_2

Materia: Plataformas de Desarrollo.
Profesor: Victor Villafañe.

Ahora estamos haciendo TP_2, no Final.

## Importante para TP_2

Tiene que ser una Single Page Application hecha solo con frontend:
- React + Vite
- HTML, CSS, JavaScript
- No backend
- No Express
- No base de datos
- Los datos pueden estar hardcodeados o en archivos mock

## Requisitos del TP_2

La app debe tener:
- login
- logout
- gestion de usuarios
- al menos 2 roles de usuario
- informacion gestionada desde interfaz grafica
- componentes separados
- buen diseño y usabilidad

No hacer una app demasiado simple tipo todo-list.

## Idea del proyecto

Proyecto para una escuela chica de idiomas.

Nombre tentativo:
AulaNova

Usuarios / roles:
- admin
- profesor
- alumno

Funciones para TP_2:
- login y logout
- dashboard diferente segun rol
- admin puede gestionar usuarios, cursos y clases
- profesor puede ver sus clases y alumnos
- alumno puede ver sus cursos y proximas clases
- datos mockeados en archivos locales
- algunas acciones de agregar, editar y borrar desde interfaz

## Pensar para el Final

Despues del TP_2, este mismo proyecto va a seguir para Final.

En el Final se agrega:
- backend separado
- Node + Express
- API REST
- base de datos
- access token
- endpoints protegidos

Por eso ahora los datos mock tienen que estar pensados para despues reemplazarse por API.

## Estilo de codigo

Codigo claro, simple, como estudiante.
No hacer arquitectura demasiado profesional.
No usar TypeScript.
No usar Redux.
No poner comentarios en cada linea.
Comentarios solo si ayudan, en español simple.
